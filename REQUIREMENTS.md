Prompt pour créer un portfolio React.js professionnel :

Crée un portfolio personnel moderne et professionnel en React.js avec les caractéristiques suivantes :

1. Design et style :

Interface moderne, minimaliste et épurée avec un thème clair et sombre (dark/light mode).

Animations fluides lors du chargement de la page, hover sur les boutons et transitions entre sections (utiliser Framer Motion ou React Spring).

Layout responsive pour mobile, tablette et desktop.

Typographie professionnelle et palette de couleurs harmonieuse, adaptée à un portfolio de développeur.

2. Sections à inclure :

Header / Hero : nom, profession, photo ou illustration animée, CTA pour télécharger le CV ou me contacter.

À propos / Bio : courte présentation avec compétences clés et expériences principales.

Compétences techniques : barre de progression animée ou badges pour HTML, CSS, JavaScript, React, Node.js, etc.

Portfolio / Projets : liste de projets avec image, titre, description, technologies utilisées, liens GitHub et démo live. Animation hover pour zoom ou transition fluide des projets.

Expérience professionnelle : timeline animée avec rôle, entreprise, période, description des missions.

Éducation : diplômes et certifications avec icônes ou animations.

Témoignages / Recommandations : slider avec avis animés.

Blog / Articles (optionnel) : afficher articles récents avec preview et bouton "Lire plus".

Contact : formulaire fonctionnel (nom, email, message), carte Google Maps et liens vers réseaux sociaux.

Footer : liens rapides, mentions légales, réseaux sociaux, copyright.

3. Fonctionnalités avancées :

Navigation sticky avec effet smooth scroll.

Lazy loading pour images et projets pour optimiser la performance.

Animation sur scroll (AOS ou Framer Motion).

Intégration API GitHub pour récupérer automatiquement les projets.

Formulaire de contact fonctionnel avec envoi par email (utiliser EmailJS ou backend Node.js).

SEO et métadonnées optimisées pour visibilité.

Mode multi-langues si possible (Français / Anglais).

4. Technologies à utiliser :

React.js avec functional components et hooks.

React Router pour navigation entre pages.

Framer Motion ou React Spring pour animations.

Styled Components ou Tailwind CSS pour stylisation.

Optionnel : TypeScript pour typage fort.

Formik + Yup pour validation du formulaire.

Axios pour appels API.

5. Animation et interactivité :

Animations au chargement et hover.

Smooth scrolling et scroll-triggered animations.

Micro-interactions sur boutons et icônes sociales.

Transitions fluides entre pages ou sections.

6. Livrable attendu :

Projet React.js complet, prêt à déployer sur Netlify / Vercel / GitHub Pages.

Code propre, structuré avec dossiers components, pages, assets, services.

README complet avec instructions d’installation et déploiement.
